# Astronomical

INFORMATIONS :

Type : Escape Game 3D
Platform : PC
Engine : Unity
Engine Version : 2022.3.4f

ABOUT THE GAME : 

Astronomical is an Escape Game which the topic is Space. 
Your mentor did some experiences and they went wrong and disturbed the current timeline of the world, you need to solve all the puzzle in your mentor's lab to close the temporal loop he openned.
